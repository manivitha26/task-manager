# Task Management App

## Features
- Create task
- View task
- Update task
- Delete task

## Tech Stack
Frontend: HTML, CSS, JS  
Backend: Node.js, Express  
Database: MongoDB  

## Setup

Backend:
cd backend
npm install
node server.js

Frontend:
Open index.html in browser

## API Endpoints
POST /api/tasks  
GET /api/tasks  
PUT /api/tasks/:id  
DELETE /api/tasks/:id
